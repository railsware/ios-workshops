Application Launch in Objects

Storyboards

Storyboard init flow

If you define a main storyboard in your project, iOS automatically:

  • Creates a new window attached to the main screen.
  • If your app delegate implements a window property, iOS sets this property to the new window.
  • Loads the main storyboard referenced in the app’s information property list file.
  • Instantiates the main storyboard’s initial view controller
  • Sets the window’s rootViewController property to the new view controller.
  • Calls the app delegate’s application:didFinishLaunchingWithOptions: method

Perform http request

  • Base: NSURL, NSURLRequest, NSURLConnection
  • Serialization: NSJSONSerialization, NSXMLParser
  • Thirdparty frameworks: AFNetworking

Demo: search tweets by name

  • NSString *baseUrl = @"https://api.twitter.com/1/users/show.json?screen_name=%@";
NSURL *url = [NSURL URLWithString:[NSString stringWithFormat: baseUrl, userName]];
NSMutableURLRequest *request = [NSMutableURLRequest requestWithURL:url];
NSData *responseData = [NSURLConnection sendSynchronousRequest:request returningResponse:nil error:nil];
NSArray *jsonData = [NSJSONSerialization JSONObjectWithData:responseData options:0 error:nil];
return jsonData;
